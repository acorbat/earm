    The Extrinsic Apoptosis Reaction Model, or EARM, is a family of models of the extrinsic apoptosis pathway written using the Python software framework PySB. Model variants focus on exploring alternative hypotheses for the mechanism of Mitochondrial Outer Membrane Permeabilization, controlled by the Bcl-2 family of proteins.

    Documentation

    All of the EARM documentation is available online (with links to source code listings) via Read the Docs.

    Download

    You can clone the source code with Git by running:

    git clone git://github.com/sorgerlab/earm

    Alternatively, you can download the code via the TAR/ZIP links above. To use the models you'll need to install PySB and its associated dependencies. More information on using the models is available in the EARM online documentation.
